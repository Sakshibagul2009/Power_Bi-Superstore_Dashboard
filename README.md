📊 Superstore Sales Performance Dashboard – Power BI
📌 Project Overview

This project presents a Superstore Sales Performance Dashboard developed using Microsoft Power BI. The dashboard provides valuable insights into sales performance, profitability, product categories, and regional performance through interactive visualizations and slicers.

The dashboard enables business users to:

Monitor overall sales and profit performance
Analyze category-wise revenue contribution
Compare regional profit performance
Track product quantity sold
Evaluate discount strategies
Make data-driven business decisions
⚙️ Real-Time Business Usefulness

Helps business leaders identify:

🧮 DAX Measures Used
Total Sales = SUM(Orders[Sales])

Total Profit = SUM(Orders[Profit])

Total Quantity = SUM(Orders[Quantity])

Average Discount =
AVERAGE(Orders[Discount])
🔹 Why Measures Are Used
Dynamic calculations based on slicers
Real-time KPI updates
Improved dashboard performance
Follow Power BI best practices
📈 Dashboard KPIs
