📊 Superstore Sales Performance Dashboard

📌 Project Overview

This project presents a Superstore Sales Performance Dashboard developed using Microsoft Power BI. The dashboard provides a comprehensive analysis of sales, profit, quantity sold, product categories, and regional performance through interactive visualizations and slicers.

The primary objective of this dashboard is to help businesses:

Monitor overall sales and profitability
Analyze category-wise performance
Compare profit across regions
Track key business KPIs
Support data-driven decision making

🎯 Dashboard Objectives

✔ Track Total Sales, Profit, Quantity, and Discount

✔ Identify the highest-performing product categories

✔ Compare regional profit performance

✔ Enable quick filtering through interactive slicers

✔ Generate actionable business insights

🛠️ Tools & Technologies Used
Microsoft Power BI
DAX (Data Analysis Expressions)
Data Visualization
Data Modeling
CSV / Excel Dataset
📈 Key Performance Indicators (KPIs)

The dashboard displays the following KPIs:

KPI	Value
Total Sales	₹1.50M
Total Profit	₹178.57K
Total Quantity	25K
Average Discount	15.70%

These KPI cards provide a quick overview of business performance.

🧮 DAX Measures Used
Total Sales = SUM(Superstore[Sales])

Total Profit = SUM(Superstore[Profit])

Total Quantity = SUM(Superstore[Quantity])

Average Discount = AVERAGE(Superstore[Discount])

Why Measures?
Dynamic calculations
Respond automatically to slicers and filters
Follow Power BI best practices
Improve dashboard performance
