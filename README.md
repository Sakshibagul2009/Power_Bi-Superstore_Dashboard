# 📊 Superstore Sales Performance Dashboard – Power BI

## 📌 Project Overview

This project presents a **Superstore Sales Performance Dashboard** developed using **Microsoft Power BI**. The dashboard provides interactive insights into sales performance, profitability, product categories, and regional performance.

The primary objective of this dashboard is to help businesses monitor key metrics, identify growth opportunities, and support data-driven decision-making through interactive visualizations and slicers.

---

## ⚙️ Real-Time Business Usefulness

### ✔ Sales Monitoring

* Track overall sales performance in real time.

* Evaluate average discounts offered to customers.

## 🧮 DAX Measures Used

```DAX
Total Sales =
SUM(Orders[Sales])

Total Profit =
SUM(Orders[Profit])

Total Quantity =
SUM(Orders[Quantity])

Average Discount =
AVERAGE(Orders[Discount])
```

### 🔹 Why Measures Are Used

* Provide dynamic calculations.
* Respond automatically to slicers and filters.
* Improve dashboard performance.
* Follow Power BI best practices.
